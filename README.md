# Chat em tempo real com Websockets em Python

## Funcionalidades
- [x] Conexão de usuários
- [x] Bate-papo em tempo real
- [x] Exibir os usuários que estão online
- [x] Identificar os usuários que mandaram mensagens
- [x] Exibir o horário em que cada um mandou mensagem
- [x] Disparar eventos quando usuários entram ou saem, ou mandam mensagens

## Features em breve
- [ ] Histórico de mensagens
- [ ] Múltiplas salas de bate-papo
- [ ] Mensagens privadas
- [ ] Upload de arquivos

## Tecnologias utilizadas
- Python 3
- Flask
- Flask-SocketIO
- HTML, CSS, JavaScript

## Objetivo
Este projeto é um chat em tempo real que utiliza Webhooks em Python.

## Como executar
1. Clone o projeto
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute: `python app.py`
4. Acesse no navegador: `http://localhost:5000`
